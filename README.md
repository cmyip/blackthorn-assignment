# Events Assignment

## Prerequisites:
* PostgreSQL (>10)
* NodeJS
* Angular `@angular/cli`

## Set up API:
- cd api
- Configure environment variables `cp .env.example .env`
- Create migration `npm run migration:generate base`
- Run migration `npm run migration:run`
- Run seed `npm run seed:run`

## Run Server
- in `api` folder run `npm run serve`
